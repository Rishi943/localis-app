## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline badge padding, chart label gaps |
| sm | 8px | Compact element spacing, tab inner padding, row gaps |
| md | 16px | Default panel padding, section content spacing |
| lg | 24px | Panel section breaks, chart vertical rhythm |
| xl | 32px | Header-to-content spacing within Finance panel |
| 2xl | 48px | Major section breaks in Dashboard |
| 3xl | 64px | Not used in Finance panel |

Exceptions:
- Transaction list rows: 9px top/bottom padding (established pattern from `.sess-item` at 9px 8px — not on 8-point grid but matches existing codebase rhythm, keep consistent)
- Bar chart track height: 8px (established from research Pattern 5)
- Touch targets on tab buttons: minimum 44px height for accessibility

**Source:** Spacing derived from existing `app.css` patterns (sessions list 9px 8px, RSB section 11px 10px, modal 16px 18px). 8-point grid applied for new Finance-specific elements.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `rgba(15, 15, 15, 0.45)` via `--bg-panel` | Finance panel background, all `.fin-*` glass surfaces |
| Secondary (30%) | `rgba(255, 255, 255, 0.05)` via `--card-bg` | Transaction list row tint, chart track background, onboarding chat bubbles, tab bar background |
| Accent (10%) | `#5A8CFF` via `--accent-primary` | See reserved list below |
| Credit indicator | `#6ee7b7` (established in app.css) | "↑ Credit" tag text and dot in transaction list only |
| Destructive | `#ef4444` via `--status-red` | "Reset goals" button hover/confirm state only |

**Accent (`--accent-primary: #5A8CFF`) reserved for:**
1. Active tab underline / active tab text
2. `.fin-bar-fill` — the filled portion of category breakdown bars
3. `.fin-dominant` bar glow (`box-shadow: 0 0 12px rgba(90, 140, 255, 0.5)`) on the single highest-spend category
4. Budget vs actual bar for the "actual" bar (budget bar uses `--card-bg` secondary)
5. "Upload CSV" primary CTA button background
6. Onboarding "Skip" link text on hover
7. Period selector dropdown focus ring

Accent is NOT used for: transaction row hover, tab backgrounds, chart track fills, body text, or general interactive states.

**No-budget state:** Budget vs actual chart shows actual bars in accent only. A subtle ghost prompt "Set up your goals to see budget tracking →" renders at `--text-muted` opacity (rgba(255,255,255,0.28)). Not a banner — it is a single line below the chart area.

**Source:** UIUX/DESIGN.md and app.css `:root` variables. Credit indicator `#6ee7b7` from existing `.tool-result .badge.green` rule in app.css line 411. Destructive `#ef4444` from `--status-red` CSS variable defined at line 59.

## Component Inventory

All new Finance components use the `.fin-` prefix. No existing class names are modified.

### Finance Panel Overlay (`.fin-panel`)

- `position: fixed; inset: 0; z-index: 200` — above RSB (z:100), below settings overlay (z:300)
- Glass recipe: `background: var(--bg-panel); backdrop-filter: var(--glass-filter); -webkit-backdrop-filter: var(--glass-filter); border: 1px solid var(--border-subtle); box-shadow: 0 16px 40px rgba(0,0,0,0.5), inset 0 1px 0 var(--border-highlight)`
- Toggled via `.hidden` class (not DOM removal) — consistent with voice-status-bar and welcome-state patterns
- Open/close animation: `opacity` + `transform: translateY(8px)` → `translateY(0)`, duration 200ms, `ease-out` — Claude's discretion, fade-slide pattern

### Tab Bar (`.fin-tabs`)

- Two tabs: "Dashboard" (default) and "Chat"
- Active tab: bottom border `2px solid var(--accent-primary)`, text `var(--text-primary)`, weight 600
- Inactive tab: text `var(--text-secondary)`, weight 400, no border
- Tab height: 44px minimum (accessibility touch target)
- Tab background: `var(--card-bg)` track, no tab-level background fill

### Period Selector (`.fin-period-select`)

- Dropdown `<select>` styled to match existing modal inputs: `background: var(--card-bg); border: 1px solid var(--card-border); border-radius: var(--r-sm); padding: 6px 12px; font-size: 13px; color: var(--text-primary)`
- Positioned at top-right of Dashboard pane header row
- Options: all uploaded `period_label` values + "All time" (default/first option)

### Category Breakdown Bars (`.fin-bar-track` / `.fin-bar-fill`)

- Track: `height: 8px; background: rgba(255,255,255,0.06); border-radius: 4px; overflow: hidden`
- Fill: `width: calc(var(--pct, 0) * 1%); background: var(--accent-primary); border-radius: 4px; transition: width 0.4s cubic-bezier(0.4, 0, 0.2, 1)`
- Dominant category only: `box-shadow: 0 0 12px rgba(90, 140, 255, 0.5)` via `.fin-dominant` class
- Category label: 12px weight 600 `var(--text-secondary)`, left-aligned
- Amount label: 12px weight 400 `var(--text-muted)`, right-aligned
- Percentage: 12px weight 400 `var(--text-muted)`, right of bar

### Budget vs Actual Bars (`.fin-budget-row`)

- Side-by-side layout: budget bar (ghost: `--card-bg` fill, `--card-border` border) + actual bar (accent fill)
- If budget = 0 or not set: show actual bar only; budget bar absent
- Over-budget state: actual bar fill switches to `--status-red` (`#ef4444`) when actual > budget
- Row label: category name at 12px/600

### Monthly Trend Bars (`.fin-trend-row`)

- One row per `period_label`, ordered by MIN(date) ascending
- Horizontal bar, same track/fill pattern as category bars but height 6px
- Period label: 11px `var(--text-muted)`, fixed-width left column 80px
- Total spend: 12px `var(--text-primary)`, right-aligned

### Transaction List (`.fin-tx-list`)

- Ghost scrollbar: extend existing selector in app.css to include `.fin-tx-list:hover ::-webkit-scrollbar-thumb`
- Row height: `min-height: 40px`, `padding: 9px 16px` (matches `.sess-item` rhythm)
- Alternating row tint: even rows `background: rgba(255,255,255,0.02)`
- Columns: Date (80px, 11px mono), Description (flex 1, 13px), Amount (72px right-align, 13px tabular-nums), Category badge, Type indicator
- Credit rows: Amount text color `#6ee7b7`, "↑ Credit" badge at 10px weight 500
- Debit rows: Amount text color `var(--text-primary)`
- Category badge: `background: var(--card-bg); border: 1px solid var(--card-border); border-radius: var(--r-sm); padding: 2px 6px; font-size: 10px; color: var(--text-muted)`

### CSV Upload UI (`.fin-upload-area`)

- Dropzone or button: glass card style (`var(--card-bg)`, `1px solid var(--card-border)`, `border-radius: var(--r-md)`)
- Period label text input: below file selector, placeholder "e.g. Jan 2026", same styling as settings inputs
- "Upload CSV" primary CTA button: `background: var(--accent-primary); color: #fff; border-radius: var(--r-pill); padding: 8px 20px; font-size: 13px; font-weight: 600`
- Upload progress: if file is large, show inline text status ("Parsing 120 rows...") — SSE optional for Phase 2, inline status acceptable
- Success state: "120 rows added, 0 duplicates skipped" in `var(--text-secondary)` below button
- Upload list: compact rows showing filename, period label, account type, row count — same `.fin-tx-list` row styling

### Onboarding Chat (`.fin-onboarding`)

- Chat-style panel inside Finance panel when `onboarding_done = false`
- Reuses `buildMessageHTML` / `appendMessage` rendering pipeline (from CONTEXT.md and RESEARCH.md)
- "Skip" button: top-right corner of panel, text link style, 13px `var(--text-muted)`, hover → `var(--accent-primary)`
- Budget input card (step 3): inline form card inside chat area — glass card (`var(--card-bg)`, `var(--r-md)`), 2-column grid of labeled number inputs, one per category, submit button at bottom
- Progress indicator: none (conversational, no step dots)

### Finance Chat Tab (`.fin-chat-pane`)

- Identical structure to main `#chat-history` but scoped to `#fin-chat-history`
- Does NOT share session with main chat — separate in-memory history array
- Input pill: reuse existing `.chat-input-row` styles, scoped with `#fin-prompt` and `#fin-send-btn`

## Interaction States

### Panel Open/Close
- Open: `financeUI.open()` removes `.hidden`, triggers `_checkOnboarding()`
- Close: `financeUI.close()` adds `.hidden`
- ESC key closes Finance panel (add keydown listener on open, remove on close)
- "Reset goals" button: inside Dashboard pane header, 11px text, `var(--text-muted)` color, no background — matches existing ghost button pattern

### Tab Switching
- Click tab → remove `.active` from current pane and tab, add to target
- No transition animation on tab content pane — instant swap (consistent with existing RSB tab pattern)

### Bar Chart Loading
- On period selector change: set `--pct` to 0 on all bars, then set to actual values after API response
- CSS `transition: width 0.4s cubic-bezier(0.4, 0, 0.2, 1)` handles the visual animation — no JS timer loop
- Loading state: bars remain at 0 width with a single pulsing opacity on the track (CSS animation, no spinner)

### Upload Flow
- File selected → period label input becomes required (red border if empty on submit attempt)
- Submit → "Uploading..." text replaces CTA button label, button disabled
- Success → button restores, success message shown, upload list refreshes
- Error → error message in `--status-red` below button, button restores

### Hover States
- Consistent with UIUX/DESIGN.md: slightly increase background opacity, do not change text color
- Transaction row hover: `background: rgba(255,255,255,0.04)` (from 0.02 base for even rows)
- Tab hover: `background: rgba(255,255,255,0.04)` on inactive tabs
