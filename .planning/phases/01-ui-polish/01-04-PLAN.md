**4. Date group separators**

Find the message list rendering function (the code that renders the full message history when a session loads). Add date grouping:

```javascript
function groupMessagesByDate(messages) {
    let lastDate = null;
    const result = [];
    for (const msg of messages) {
        const msgDate = new Date(msg.created_at || Date.now());
        const dateStr = msgDate.toLocaleDateString('en-US', { weekday: undefined, month: 'short', day: 'numeric' });
        const today = new Date(); const yesterday = new Date(); yesterday.setDate(today.getDate() - 1);
        const isSameDay = (a, b) => a.toDateString() === b.toDateString();
        const label = isSameDay(msgDate, today) ? 'Today' : isSameDay(msgDate, yesterday) ? 'Yesterday' : dateStr;
        if (label !== lastDate) {
            result.push({ type: 'separator', label });
            lastDate = label;
        }
        result.push(msg);
    }
    return result;
}
```

When rendering messages in the chat history, iterate through `groupMessagesByDate(messages)` and insert a `<div class="date-separator">${item.label}</div>` when `item.type === 'separator'`. This only applies to the history load render, not to streaming messages (a streaming message always belongs to "Today" — no need to inject a separator for each token).
