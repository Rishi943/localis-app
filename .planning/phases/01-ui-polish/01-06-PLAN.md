  <how-to-verify>
Start the app: `uvicorn app.main:app --reload --host 0.0.0.0 --port 8000` then open http://localhost:8000

Walk through each requirement:

**UI-01 — Layout and spacing:**
1. Look at the three-column layout — LSB, main chat, RSB. Do panels feel evenly padded and aligned?
2. Set browser to 1440x900. Does the layout fit naturally without needing to zoom to 110%? (Sidebar widths should widen via clamp())
3. Set browser to 1920x1080. Does the chat column stay centered and readable (not full-width)?

**UI-02 — Navigation and sidebars:**
4. Click the left sidebar collapse button. Does it animate smoothly to 44px (icons only)?
5. When LSB is collapsed, is the gear icon still visible and clickable at the bottom?
6. Click the gear icon — does the settings modal open reliably? Click it 3 times to confirm.
7. In settings modal, switch tabs (Inference, Appearance, Profiles). Do all three tabs render correctly?
8. Select a profile in the Profiles tab — does the RSB chip highlight update immediately?
9. Click the RSB toggle strip (right edge) — does the RSB collapse/expand correctly?
10. Load a session with previous messages — is the session history list styled and scrollable?

**UI-03 — Chat surface:**
11. Type a message and send it. Does the chat auto-scroll as the response streams in?
12. After the response completes, do Copy / Regenerate / Continue chips appear below the message?
13. Click Copy — is the message text copied (no error in console)?
14. Enable wakeword ("Hey Jarvis") — does the voice status bar appear as a small, faint indicator (not a big pill)?
15. Type a long message (100+ chars). Does a faint "~N tokens" appear in the input area?
16. Open a fresh session with no messages — does the welcome area show with suggestion chips?
17. Click a suggestion chip — does it populate the input and hide the welcome area?

**UI-04 — Visual cohesion:**
18. Look at the LSB, RSB, and settings modal — do they all share the same glass surface treatment (dark, translucent, subtle rim highlight)?
19. Is the body background pure black (no colored glow or radial gradient)?
20. Are assistant messages labelled "Localis" (not "Jarvis")?
21. Does the RSB show "LIGHTS", "STATS", "MODEL" section labels? Are there subtle 1px dividers between them?
22. Do CPU, RAM, VRAM stats show as compact bar + number rows (not a wall of text)?
