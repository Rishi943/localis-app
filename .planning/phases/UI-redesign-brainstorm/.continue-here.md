<decisions_made>

## Input Area Layout — Option C: Action Cluster + Minimal Pill
- Tools (web toggle, think toggle, remember, upload) + Voice (mic on/off, wakeword on/off) live in a labeled cluster panel ABOVE the input pill
- The pill itself is minimal: just text input + send button
- Cluster has two groups: "TOOLS" | "VOICE" separated by a vertical divider
- Each button shows icon + label text
- Active/on states: web/think/remember = blue (rgba(18,117,226) tint); mic on = green; wakeword on = yellow
- Voice status bar ("Hey Jarvis") ONLY shows when wakeword toggle is ON, hidden otherwise

## Voice Status Bar
- Only visible when wakeword is toggled ON
- 3 states: idle gray ("Say 'Hey Jarvis' to activate") → amber pulsing ("Listening…") → green ("Done", 2s then reset)
- Pill shape, centered above the action cluster

## Left Sidebar (LSB) — Option A: Icon Strip when collapsed
- Open state: matches Stitch screenshot exactly (brand icon + "Localis" / "MIDNIGHT GLASS AI", New Chat button, History list, model bar at bottom with status dot + gear)
- Collapsed state: 64px wide icon strip — brand icon (bolt), chevron expand button, chat/history icons, model status dot + gear at bottom
- Toggle button: small circle on right edge of LSB (absolute positioned), chevron rotates on collapse
- Width transition: 260px ↔ 64px, smooth cubic-bezier

## Right Sidebar (RSB) — Single Combined Scrollable Panel
- Collapsed: 64px icon strip showing EXACTLY 4 icons: Home Automation (home_iot_device), Model Loader (memory), Finance Dashboard (payments), Notes & Reminders (description) + help/settings at bottom
- Clicking HA or Model icon when collapsed → snap-scroll to that section in expanded panel
- Finance and Notes icons → open their existing full-screen overlays (no inline panel)
- Expanded: Single scrollable "Quick Controls" panel with 4 sections separated by dividers:
  1. HOME AUTOMATION (lights)
  2. MODEL (loader)
  3. SYSTEM PROMPT
  4. STATS
- Close button: chevron_right icon (consistent across all panels/modals)

## RSB — Home Automation Section
- Light name + toggle switch + "X min ago" timestamp
- Bulb graphic (80×130px, rounded rect, fill animates with brightness, drag line)
- % readout below bulb
- 4 control buttons (circular): power, brightness (active=amber highlight), color palette, thermometer/kelvin
- 8 color swatches in a row
- Brightness slider (0%–100%) with amber thumb

## RSB — Model Loader Section
- Model name in JetBrains Mono + ONLINE/OFFLINE status
- Arrow (expand_more) toggles the model list dropdown open/closed — collapsible, NOT always shown
- Arrow rotates 180° when open
- Model list items in JetBrains Mono, selected item has blue tint + checkmark
- Load (blue) / Unload (red tint) action buttons
- GPU Layers slider below

## RSB — System Prompt Section
- 2×2 grid layout for preset chips (Default | Creative / Planning | Custom) — NO staggered wrapping
- Edit System Prompt row with edit_note icon + arrow_forward

## RSB — Stats Section
- CPU (green bar), RAM (indigo bar), VRAM (amber bar)
- Each: label left, value right (%, GB), thin 3px bar below

## Chat Messages — Tool Use Cards
- Tool invocations shown as collapsible "pill" above the AI bubble
- Pill: status dot + icon + "tool.name · done/N results" + chevron
- Click pill to toggle detail card open/closed
- Collapsed by default for older messages; expanded by default for most recent
- Light action card: shows light name, entity ID, change badge (→ ON / → OFF)
- Memory card: blue tint border, bookmark_added icon, key/value pairs in monospace keys
- Web search card: numbered results with title (accent-light), snippet, domain (mono)

## Chat Bubbles
- User: rounded pill `18px 18px 4px 18px`, `rgba(255,255,255,0.08)` bg, max-width clamp(200px, 70%, 520px)
- AI: rounded `4px 18px 18px 18px`, glass panel, max-width clamp(280px, 100%, 680px)
- Avatar: user = blue/indigo gradient circle; AI = dark circle with bolt icon (accent-light)
- Message actions (below AI bubble): copy icon, refresh icon, thumbup icon + token stat in mono

## Thinking Block
- Collapsible, collapsed by default
- Header: psychology icon + "Reasoning" label + "Xs · click to expand" hint + chevron
- Body: italic text, dark bg, smooth max-height transition

## Responsive
- All widths use `clamp()` for bubble max-widths
- Action cluster wraps gracefully (it's flex, not fixed width)
- Tool cards: `max-width: clamp(260px, 100%, 380px)`
- At narrow widths the tool indicator truncates with ellipsis

## Design System (from UIUX/DESIGN.md — unchanged)
- Identity: "Midnight Glass" — pure black + liquid glass
- Fonts: Inter (UI), JetBrains Mono (code/mono values)
- CSS vars: --bg-app:#000, --bg-sidebar:rgba(10,10,10,0.55), --bg-panel:rgba(15,15,15,0.45)
- Glass recipe: backdrop-filter:blur(24px) saturate(180%), border:1px solid rgba(255,255,255,0.15)
- Accent: #1275e2 / --accent-light: #aac7ff
- Scrollbars: 3px wide, rgba(255,255,255,0.07–0.10), transparent track

## Consistency Rules
- All close/collapse buttons: chevron_right icon in 28×28px rounded rect (rgba(255,255,255,0.05) bg, rgba(255,255,255,0.08) border)
- All back buttons: chevron_left (same style)
- Section labels: 9px uppercase, letter-spacing 0.13em, rgba(255,255,255,0.25), with ::after line
- Dividers between RSB sections: 1px rgba(255,255,255,0.05)
